Getting Started

            
Agent Engineer allows you to create intelligent agents using simple configurations. Select a paradigm, configure your agent, and generate LangGraph code automatically.

            
            
Paradigms

            
    
                
  • React: Best for single-agent sequential tasks
    • 
                
  • Surprise: Ideal for adaptive learning systems
    • 
                
  • Peer: Great for collaborative problem-solving
    • 
                
  • Hierarchical: Perfect for complex workflows
    • 
                
  • Evolutionary: Best for optimization problems
    • 
            

            
            
API

            
Use the REST API to integrate agent generation into your applications:

            
POST /api/generate
{
  "name": "My Agent",
  "description": "An agent that...",
  "paradigm": "react",
  "tools": ["search", "browser"]
